# Vellum — Landing Page

架空のライティングアプリ **Vellum** のランディングページ（デザインサンプル）です。デザインは **Y2K・レトロポップ** 路線。

🔗 **公開URL:** https://gray-desks.github.io/vellum-lp/

## 特徴

- **Y2K・レトロポップ** — 原色（ライム / ホットピンク / エレクトリックブルー）＋ハードシャドウ
- 極太丸ゴシック `Dela Gothic One` / `Mochiy Pop One` / `Zen Maru Gothic`
- 回転ステッカー、クローム文字、流れるマーキー、カーソルのキラキラ軌跡、スクロール連動のポップ表示
- 依存ゼロの単一HTML。`index.html` を開くだけで動きます

## ローカルで見る

```bash
open index.html
# もしくは
python3 -m http.server 8000   # → http://localhost:8000
```
