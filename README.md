# Vellum — Landing Page

思考のための、静かな書く場所。架空のライティングアプリ **Vellum** のランディングページ（デザインサンプル）です。

🔗 **公開URL:** https://gray-desks.github.io/vellum-lp/

## 特徴

- **モダン・ミニマル × エディトリアル** — 温かみのある紙色 + インク黒 + テラコッタの一点差し
- 表示書体 `Fraunces`、本文 `Instrument Sans`、ラベル `JetBrains Mono`
- 微細なグレインテクスチャ、読み込み時のスタッガー表示、スクロール連動アニメーション
- 依存ゼロの単一HTML。`index.html` を開くだけで動きます

## ローカルで見る

```bash
open index.html
# もしくは
python3 -m http.server 8000   # → http://localhost:8000
```

## 構成

| セクション | 内容 |
|---|---|
| Hero | キャッチコピー + エディタ風モック |
| Features | 引き算でつくる3つの機能 |
| Craft | 細部へのこだわり |
| Pricing | 3プランの料金表 |
| CTA | 早期アクセス登録フォーム |
